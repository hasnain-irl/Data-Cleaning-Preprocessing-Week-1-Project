# Data Cleaning & Preprocessing – Week 1 Project

## 📌 Overview

This project focuses on cleaning and preparing a raw dataset as part of my Data Analytics Internship at DecodeLabs.

The goal was to ensure the dataset is accurate, consistent, and ready for analysis.


## 🎯 Objectives

* Identify and handle missing values
* Remove duplicate records
* Standardize inconsistent data formats (dates, text)

---

## 🛠️ Tools Used

* Microsoft Excel

---

## 🔍 Data Cleaning Steps

### 1. Handling Missing Values

* Replaced empty values in the "Coupon Code" column with `"NoCode"`
* Total affected cells: 309

### 2. Standardizing Date Format

* Converted all date values to ISO 8601 format (YYYY-MM-DD)

### 3. Data Validation

* Checked for duplicates and inconsistencies
* Ensured uniform structure across the dataset


## 📈 Outcome

* Cleaned and structured dataset ready for analysis
* Improved data consistency and interpretability
* No advanced tools required due to good initial data quality


## 💡 Key Learning

Even simple datasets require careful inspection. Data cleaning is a critical step that directly impacts the accuracy of analysis and insights.
